### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   bun install
   ```

3. Set up environment variables:
   ```bash
   cp .env.example .env
   # Edit .env with your configuration
   ```

4. Push database schema:
   ```bash
   bun run db:push
   ```

5. Seed the database with sample data:
   ```bash
   bun seed.ts
   ```

6. Start the development server:
   ```bash
   bun run dev
